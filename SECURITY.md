# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities by email:

[security@deasilsoft.com](mailto:security@deasilsoft.com)

Do not report security vulnerabilities through public GitHub issues.

## Supported Versions

Security fixes are provided for the latest published version of `code-slicer`.

## Response

We will review valid security reports as soon as reasonably possible.

If the issue is confirmed, we will coordinate a fix and release before public
disclosure where appropriate.
